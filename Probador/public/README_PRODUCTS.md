# Product Images Instructions

This folder should contain 4 product images:

1. **product1.jpg** - Premium Athletic Shorts (black with side stripes)
2. **product2.jpg** - Professional Training Leggings (anthracite gray)
3. **product3.jpg** - Elite Performance Tank (pure white)
4. **product4.jpg** - Advanced Compression Top (royal blue, long sleeve)

## Image Requirements:
- Format: JPG or PNG
- Recommended size: 400x600px minimum
- Background: Clean, preferably white or light gray
- Content: Clear product shot showing the garment

## How to add images:
1. Name your product images exactly as listed above
2. Place them in this `public/products` folder
3. The application will automatically use them as references for the AI generation

Note: The AI will use these images as reference "B" to generate the virtual try-on, combining the user's photo with the actual product image you provide.
